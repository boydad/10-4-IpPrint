Подробное описание

Содержит семейство функций printIp, которые печатают ip побайтно в поток os, разделяя байты точкой.

Семейство функций printIp(os, ip) принимает поток вывода std::ostream& os, куда осуществляется вывод, и ip адресс, который печатается в os. Функции ничего не возвращают.

Функции перегруженны для следующих типов ip адреса:

  • базовые типы;
  • строка std::string (печатается как есть;
  • контенеры std::vestor и std::list (печать происходит поэлементно);
  • кортеж std::tupl (печатается поэлементно).

Примеры ипользования приведены в main().
